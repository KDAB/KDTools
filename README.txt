Introduction
============
KD Tools aims to ease the daily work of Qt programmers by providing a number
of well-designed, easy-to-use widgets and non-gui classes.

KDAB provides no promises of support or maintenance for this code.

Using KD Tools
==============
KD Tools requires Qt4.x.  At this time KD Tools does not build with Qt5.

See INSTALL.txt for installation instructions.

You will find more information in these places:

  detailed browsable API reference:   http://docs.kdab.com/kdtools
  our sorted example programs:        examples/
